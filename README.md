# TUCM

TUCM是一个优秀的人造语言（conlang）社区，致力于造语爱好者的交流与合作。

## 项目介绍

造语者联盟（TUCM）官方网站，提供造语相关的资源、工具和社区信息。

## 历史沿革

- 2020年5月17日，造语联盟成立
- 2020年8月8日，造语联盟贴吧成立
- 2020年下半年，造语联盟QQ群更名为造语者联盟，标志着联盟正式成立
- 2023年10月5日，建立造语者联盟Fandom Wiki
- 2025年2月21日，自建官网上线

## 功能与资源

### 内容板块

- 汝且候简介：创始人介绍
- 造语者联盟历史：详细的社区发展历程
- 文集：造语相关的文章和作品

### 工具

- 造词机：辅助造词的工具
- 查索玛苏语词典：苏玛苏语的在线词典
- 随机人名地名：生成随机的人名和地名
- 在线词典：词典管理小工具

### 技术栈

- 前端：HTML, CSS, JavaScript
- 无后端依赖，纯静态网站
